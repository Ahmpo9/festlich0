Häufige Fragen

Alles, was du wissen möchtest

Wie lange bleibt meine Einladung online?+
Deine Einladung bleibt so lange aktiv, wie dein Konto besteht. Alle bezahlten Pläne beinhalten unbegrenzten Zugang — du zahlst einmal und die Einladung ist dauerhaft erreichbar.
Kann ich eigene Fotos hochladen?+
Ja! Du kannst eigene Verlobungs- oder Hochzeitsfotos hochladen und direkt im Editor positionieren, zoomen und ausrichten. Die Einstellungen werden pro Bild in der Datenbank gespeichert.
Funktioniert der QR-Code auch auf GitHub Pages?+
Ja, das ist eines unserer wichtigsten Features. Der QR-Code und alle Einladungslinks nutzen das Format festlich-invitation.html?slug=... — damit funktionieren sie auf GitHub Pages, Netlify und jedem anderen statischen Host, ganz ohne Server.
Können Gäste von jedem Gerät aus antworten?+
Absolut. Festlich ist mobile-first entwickelt. Kein App-Download nötig — die Einladung öffnet sich direkt im Browser, auf jedem Smartphone, Tablet oder Computer.
Kann ich die Einladung nach dem Teilen noch bearbeiten?+
Ja, jederzeit. Alle Änderungen erscheinen sofort auf der Live-Einladung — der Link und QR-Code bleiben dabei unverändert. Deine Gäste müssen nichts neu öffnen.
Welche Sprachen unterstützt Festlich?+
Das Dashboard ist vollständig auf Deutsch und Englisch verfügbar. Du kannst außerdem für jede Einladung einen deutschen und einen englischen Willkommenstext eingeben.
Gibt es eine Rückgabegarantie?+
Wir bieten eine vollständige Rückerstattung innerhalb von 14 Tagen, keine Fragen gestellt. Kontaktiere uns einfach per E-Mail an hallo@festlich.de.
