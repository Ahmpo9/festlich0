FAQ

Häufige Fragen

Kann ich wirklich kostenlos gestalten? +

Ja. Registrieren, Design wählen, gestalten — alles kostenlos. Du bezahlst erst die 39,99 €, wenn du zufrieden bist und die Einladung herunterladen möchtest.

Was bekomme ich nach dem Bezahlen? +

Sofortigen Download als hochauflösendes Bild (für WhatsApp & Co.), PDF (für Druck) und einen persönlichen Teilen-Link mit QR-Code.

Kann ich mein eigenes Foto nutzen? +

Absolut. Lade dein Verlobungs- oder Lieblingsfoto hoch, positioniere und zoome es frei im Editor.

Funktioniert das auf GitHub Pages? +

Ja. Festlich ist komplett statisch und funktioniert auf GitHub Pages, Netlify und jedem anderen Hosting — ohne Server.

Gibt es eine Geld-zurück-Garantie? +

Ja. Wenn du nicht zufrieden bist, erstatten wir den vollen Betrag innerhalb von 14 Tagen. Ohne Fragen.
